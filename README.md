# Expense Tracker

## Overview
This project contains a backend API (Node/Express/MongoDB) and a frontend SPA (React + Vite). Follow the instructions below to run locally.

### Backend (run in `backend/`)

1. Copy `.env` and set `MONGO_URI` and `PORT`.
2. Install dependencies:
   cd backend
   npm install
3. Start server (development):
   node server.js
4. The API runs at `http://localhost:4000` 

### Frontend (run in `frontend/`)
1. Install dependencies:
   cd frontend
   npm install
2. Start dev server:
   npm run dev
3. Open the URL printed by Vite
